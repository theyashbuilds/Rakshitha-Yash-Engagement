# Yash & Rakshitha Engagement

A beautiful engagement ceremony invitation website for Yash and Rakshitha's special day on 25th January 2026.

## Technologies Used

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Getting Started

Make sure you have Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

```sh
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Project Structure

- `/src` - Source code
- `/public` - Static assets
- `/index.html` - Entry HTML file
